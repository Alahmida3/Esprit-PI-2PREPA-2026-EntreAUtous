# Esprit-PI-2PREPA-2026-EntreAUtous - Plateforme de Gestion Automobile

## Description du Projet
entreAUtous est une plateforme web qui permet aux propriétaires de voitures de gérer facilement leurs véhicules et leurs services automobiles.

Elle permet de :
- Gérer les véhicules 
- Prendre des rendez-vous 
- Connecter les clients avec les garages 
- Organiser les services automobiles 

Objectif : simplifier et digitaliser la gestion automobile.
